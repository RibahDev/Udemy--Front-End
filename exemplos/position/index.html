static

Static é a posição padrão para todos os elementos sem alterações elementos com esse valor não são afetados por top, left, right e bottom

eu sou statico

Relative

Relative é afetado por comandos como top, left, right e bottom. Porem não utilizamos muito essa combinação com Relative!

Eu sou Relativo

Exemplo de o porque NÃO usamos relative em muitos casos


Absolute

Absolute também é afetado por top, left, right e bottom. É possivel movimentar o elemento com Absolute por toda a tela, ele quebra o fluxo do HTML

Eu sou absoluto

Exemplo do porque pode ser um problema ou uma solução...

como se não existisse para o html ou como se mudasse de plano, vemos mas não interfere em posicionamentos alheios


Sticky

Funciona como uma mistura de fixed + relative, fazendo com que o elemento fique na tela mas a partir do momento que passamos do posicionamento decretado ao elemento ele acompanha a tela como vemos em fixed.

Eu sou sticky!

Absolute + Relative

Quando possuimos um elemento pai relativo e os elementos filhos com position absolute, tomamos como base o posicionamento do relativo e não o body em si.

Fixed

Estou fixo ;)

Fixed posiciona um elemento fixo em sua tela, rolando a pagina para cima ou para baixo o elemento te siguirá
